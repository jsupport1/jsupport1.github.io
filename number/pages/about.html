Our Mission

Technology Stack

DailyNumberTask is a static Progressive Web App — no build step, no framework, no CDN for the core logic. The entire tool ships in under 30KB of gzip-compressed assets.

  • Vanilla JavaScript (ES Modules) — zero framework overhead, native module caching
  • BigInt API — arbitrary-precision integer arithmetic in the browser
  • Service Worker + Cache API — offline-first, cache-first for assets
  • Web App Manifest — installable as a native-like PWA on Android & iOS
  • Express.js static server — thin Node.js layer for clean URLs and security headers
  • Semantic HTML5 + ARIA — targeting WCAG 2.1 AA compliance
